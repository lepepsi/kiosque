# kiosque
es handelt sich um die Entwicklung eines Verwaltungssystems für einen Kiosk.
In diesem Kiosk werden Mahlzeiten wie: Omelette, Spaguetti, Reis, begleitet von verschiedenen Würzmitteln wie: Wurst, Sardine und Schinken verkauft. 
Dieses System wird es uns ermöglichen, die im Laufe des Tages getätigten Verkäufe zu speichern und zu analysieren. Die Verkäufe können sofort konsultiert werden.
